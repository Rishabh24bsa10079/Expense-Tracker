EXPENSE TRACKER 

made by Rishabh Kumar Sudhansu[24BSA10079]



Purpose
- Simple command-line expense tracker used for learning Java IO, collections and basic design.

Quick start (Windows)
1. Open a terminal in the project folder:
   cd "c:\Users\goura\Java\ExpenseTracker"
2. Compile:
   javac *.java
3. Run:
   java Main
4. Data is stored in `expenses.csv` in the same folder.

Notes
- CSV storage is intentionally simple for classroom assignment.
- This project is small and readable; use the source and design doc to explain choices.
